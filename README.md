PAF 前端模块化组件库
=====================

###简介

做有节操的组件库，遵守PAF项目的UI风格，提供简单方便的API，并且实现最轻量，最实用的代码。


###插件列表
* [Dialog](/mod/dialog)
* [Tips](/mod/tips)
* [Mask](/mod/mask)
* [Alert](/mod/alert)
* [Tabs](/mod/tabs)
* [Template](/mod/template)
* [FormValid](/mod/formValid)
* [Util](/mod/util)



###注：
* 所有的组件遵循模块化书写方式, 并可直接移植至nodejs环境
* 所有的组件未添加AMD或者CMD或者其他规范的组件头部，使用者可自行选择以合适的方式添加合适的头部。
